这是一个专门为医疗诊断和治疗方案匹配开发的系统，旨在提供高效、准确的医疗决策支持。

系统要求

  • Python 3.7+
  • tkinter
  • matplotlib
  • numpy

安装说明

  1. 克隆仓库：
    git clone https://github.com/zhanshaodong/DIP2.0.git
  2. 安装依赖：
    pip install -r requirements.txt
  3. 运行程序：
    python medical_matcher/main.py
